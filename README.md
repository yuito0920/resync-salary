# resync スタッフ給与計算システム

## 🌐 アクセス方法

https://yuito0920.github.io/resync-salary/

## 📱 使い方

1. ブラウザで上記URLにアクセス
2. トレーナーを選択してPINを入力（初期PIN: 0000）
3. セッション回数を入力すると自動で給与計算されます

## ⚙️ 機能

- ✅ PIN認証によるトレーナー別管理
- ✅ セッション回数入力
- ✅ 自動給与計算（広告/自己集客/モニター対応）
- ✅ チケット（コース）管理
- ✅ 月次レポート表示
- ✅ Excel出力機能
- ✅ データバックアップ/復元

## 🔐 セキュリティ

- データはブラウザのローカルストレージに保存されます
- 各トレーナーは自分のデータのみ閲覧可能
- 管理者PINで全トレーナーのデータを管理できます
